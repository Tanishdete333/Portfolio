# 🌐 Personal Portfolio

> A modern developer portfolio showcasing my full-stack projects, technical skills, and journey as a software engineer.

## Overview

This portfolio serves as the central hub for my work as a full-stack developer. It brings together the projects I've built, the technologies I work with, and my ongoing journey as a Computer Science student specializing in AI & Data Science.

Rather than focusing on tutorial exercises, this portfolio highlights production-ready applications that I've designed, developed, and deployed. Each featured project demonstrates practical experience with frontend development, backend architecture, REST APIs, databases, authentication, and cloud deployment.

## Featured Projects

### VIP Number Wala

Production e-commerce platform for browsing and booking premium mobile numbers, complete with cart functionality, admin dashboard, and Supabase backend.

### NoteWall

Full-stack sticky note application with Clerk authentication, private user workspaces, MongoDB Atlas, and REST API.

### Photo Feed

Full-stack photo sharing platform featuring image uploads through ImageKit, MongoDB storage, and a responsive React interface.
